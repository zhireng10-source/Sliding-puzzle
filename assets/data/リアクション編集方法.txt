====================================
リアクション吹き出しの編集方法
====================================

リアクション吹き出しのセリフは、簡単に編集できます。

【編集するファイル】
  js/reactions-data.js

【編集方法】
  1. テキストエディタ（メモ帳など）でreactions-data.jsを開く
  2. セリフを自由に編集する
  3. 保存する
  4. ゲームを再起動する

【注意事項】
  ✅ 日本語が使えます
  ✅ セリフは「"」（ダブルクォート）で囲んでください
  ✅ セリフの間は「,」（カンマ）で区切ってください
  ✅ 最後のセリフの後にはカンマは不要です
  ❌ JavaScriptの構造を壊さないように注意してください

【ファイル構造】
  window.REACTIONS_DATA = {
    "reactions": [
      通常のリアクション（250個程度）
    ],
    "doubleTapReactions": [
      同じ場所を連続でクリックした時のリアクション（15個）
    ],
    "rapidReactions": [
      高速連打した時のリアクション（15個）
    ]
  };

【編集例】
  変更前：
    "「やだっ、チームメイトに見られたら…！」",

  変更後：
    "「きゃっ、恥ずかしい…」",

【セリフの追加方法】
  既存のセリフの後に、カンマで区切って追加できます：

  "「既存のセリフ」",
  "「追加したいセリフ1」",
  "「追加したいセリフ2」",

【セリフの削除方法】
  削除したいセリフの行を削除します。
  削除後、前の行の最後にカンマがあるか確認してください。

【トラブルシューティング】
  - セリフが表示されない場合
    → JSONファイルの構文エラーの可能性があります
    → ブラウザのコンソール（F12キー）でエラーを確認してください

  - ゲームが動かない場合
    → reactions.jsonをバックアップから復元してください
    → カンマ、ダブルクォートの位置を確認してください

【バックアップ推奨】
  編集前に必ずreactions.jsonのバックアップを取ってください！
